\## 1. 対数微分法 (対数微分法)

基準:



式が Pow (べき乗) の形 A\*\*B である。



かつ、底 A と 指数 B の両方が変数 x を含んでいる (.has(x) が True)。



例: x\*\*x, x\*\*sin(x), (x+1)\*\*(x+2)



\## 2. 積の微分 (積の微分)

基準:



式が Mul (掛け算) の形 A \* B \* C ... である。



かつ、掛け合わされている項 (A, B, C...) のうち、指数が負の数でない項の中で、変数 x を含む (.has(x) が True) ものが 2つ以上存在する。



例: x\*sin(x), exp(x)\*cos(x), x\*\*2\*(x+1)



除外例: 5\*x\*\*2 (xを含む項が1つのため)



\## 3. 商の微分 (商の微分)

基準:



式が Pow (べき乗) の形 A\*\*B である。



かつ、指数 B が 負の数 (.is\_negative が True) である。



かつ、底 A が変数 x を含んでいる (.has(x) が True)。



例: (x+1)\*\*-1 ( 1/(x+1) ), sin(x)\*\*-2



補足: x/sin(x) は Sympy によって x \* sin(x)\*\*-1 と解釈されるため、「積の微分」と「商の微分」の両方に該当します。



\## 4. 三角関数の微分 (三角関数の微分)

基準:



式が関数呼び出しの形 f(...) であり (.func 属性を持つ)。



かつ、その関数 f が sin, cos, tan のいずれかである。



例: sin(x), cos(x\*\*2), tan(x+1)



\## 5. expの微分 (expの微分)

基準:



式が関数呼び出しの形 f(...) であり (.func 属性を持つ)。



かつ、その関数 f が exp である。



例: exp(x), exp(2\*x)



\## 6. 対数関数の微分 (対数関数の微分)

基準:



式が関数呼び出しの形 f(...) であり (.func 属性を持つ)。



かつ、その関数 f が log である (文字列 "ln" も log として解釈される前提)。



例: log(x), log(x\*\*2+1), ln(x)



\## 7. べき乗微分 (x^n) (べき乗微分 (x^n))

基準:



式が Pow (べき乗) の形 A\*\*B である。



かつ、底 A が Symbol('x') (単純な変数 x) である。



かつ、指数 B が 整数 (.is\_Integer が True) である。



例: x\*\*2, x\*\*5



除外例: (x+1)\*\*2 (底が x でない), x\*\*2.5 (指数が整数でない)



\## 8. ルートの微分 (ルートの微分)

基準:



式が Pow (べき乗) の形 A\*\*B である。



かつ、底 A が変数 x を含んでいる (.has(x) が True)。



かつ、指数 B が 数値 (.is\_Number) であるが、整数 (.is\_Integer) ではない (例: 0.5, 1/3)。



(※ sqrt(x) は Sympy によって x\*\*0.5 または Pow(x, Rational(1, 2)) と解釈されるため、この基準に含まれます。)



例: x\*\*0.5, (x\*\*2+1)\*\*(1/3), sqrt(x+1)



\## 9. 合成関数の微分 (合成関数の微分)

基準:



式が sin, cos, tan, exp, log, Pow, sqrt いずれかの関数または演算の形である。



かつ、その中身（引数 .args）のいずれかが、単純な変数 Symbol('x') ではなく、x を含む別の式 (.has(x) が True かつ Symbol でない) である。



ただし、単純な x\*\*n (nは整数) の形は除く (これは「べき乗微分」で処理されるため)。



例: sin(x\*\*2) (中身が x\*\*2), exp(2\*x) (中身が 2\*x), (x+1)\*\*5 (底が x+1), log(x+1) (中身が x+1)



除外例: sin(x) (中身が x), x\*\*5 (単純なべき乗)
